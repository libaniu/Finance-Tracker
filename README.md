💸 Myllet
Personal Finance Tracker + AI Advisor. Aplikasi pencatat keuangan mobile-first yang ditenagai oleh Google Gemini AI & Supabase.

✨ Fitur Utama
🤖 AI Advisor: Analisa keuangan & saran hemat cerdas via Gemini AI.
📝 Quick Tracking: Catat pemasukan/pengeluaran dengan cepat.
📊 Budgeting: Set target bulanan & pantau progress real-time.
☁️ Cloud Sync: Data tersimpan aman di Supabase (PostgreSQL).
📱 PWA Support: Bisa di-install di HP (Add to Home Screen).
🛠️ Tech Stack
Core: Next.js 14 (App Router), TypeScript
Styling: Tailwind CSS, Lucide React
Backend/DB: Supabase
AI: Google Gemini API
